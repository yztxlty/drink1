# Code of Conduct

## Our Pledge

We pledge to make participation in Drink1 a respectful, inclusive, and harassment-free experience for everyone, regardless of background, identity, experience level, or role.

## Expected Behavior

- Be respectful in issues, pull requests, comments, and reviews.
- Focus feedback on ideas and implementation, not on people.
- Assume good intent and communicate clearly.
- Be constructive when pointing out problems.
- Make room for different viewpoints and learning levels.

## Unacceptable Behavior

- Harassment, discrimination, or demeaning language
- Personal attacks, insults, or repeated disruption
- Publishing private information without permission
- Intimidation, stalking, or inappropriate contact
- Any behavior that makes collaboration unsafe or hostile

## Enforcement Responsibilities

Project maintainers are responsible for interpreting and enforcing this code of conduct. They may remove, edit, or reject contributions that violate these standards.

## Reporting

If you experience or witness behavior that violates this code of conduct, please report it to the repository maintainer through a private channel.

When reporting, include:

- A short description of what happened
- The approximate time and place
- Any relevant links, screenshots, or commit references
- Any immediate safety concerns

## Enforcement Process

1. The report is reviewed privately.
2. The maintainers assess the situation and decide on an appropriate response.
3. Responses may include a warning, temporary restriction, or permanent removal from project participation.
4. The goal is to restore a safe and productive environment, not to punish good-faith mistakes.

## Scope

This code of conduct applies to all project spaces, including GitHub issues, pull requests, code reviews, release discussions, and any project-related communication.
